== Description ==

FraudLabs Pro is a fraud prevention add-on for WP e-Commerce to help merchants to protect their online stores from malicious fraudsters by screening all order transactions for fraud patterns. Its comprehensive and advanced algorithm engines validate all elements such as geolocation, proxy, email, blacklist, credit card, transaction velocity and others to unveil fraud orders accurately. This add-on operates behind the scene without interrupting the payment process and it provides detailed reports of all orders for the merchant’s reference.

== Installation ==

1. Upload `fraudlabs-pro` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Enter the License Key at the 'Settings' section.
